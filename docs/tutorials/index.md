# Tutorials

Learn ytofu through hands-on tutorials.

## Overview

These tutorials guide you through building real infrastructure step by step. Each tutorial builds on previous concepts.

## Skill Levels

| Level | Description | Time |
|-------|-------------|------|
| Beginner | New to ytofu or infrastructure as code | 15-20 min |
| Intermediate | Familiar with basic concepts | 30-45 min |
| Advanced | Ready for production patterns | 45-60 min |

## Prerequisites

Before starting, ensure you have:

1. **ytofu installed** - See [Getting Started](../getting-started.md)
2. **AWS credentials configured** - `aws configure` or environment variables
3. **Basic YAML knowledge** - Indentation and syntax

## Tutorial Format

Each tutorial includes:

1. **Goal** - What you'll build
2. **Prerequisites** - What you need before starting
3. **Step-by-step instructions** - Detailed walkthrough
4. **Complete code** - Full configuration to reference
5. **Cleanup** - How to destroy resources

## After the Tutorials

Once you've completed the tutorials:

- **Explore patterns** - See [Guides](../guides/index.md) for production patterns
- **Reference docs** - Find specific resources in [AWS Reference](../aws/index.md)
- **Understand concepts** - Deep dive in [Concepts](../concepts/index.md)
